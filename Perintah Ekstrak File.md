=== > unzip

01. Ekstrak file .zip ke folder saat ini.

    Perintah : unzip [nama_file.zip]
	Fungsi   : Perintah tersebut akan mengekstrak isi file .zip ke folder saat ini.

02. Ekstrak file .zip ke folder lain.

	Perintah : unzip [nama_file.zip] -d lokasi/folder/output
	Fungsi   : Perintah tersebut akan mengekstrak isi file .zip ke folder yang akan dituju.

03. Melihat isi file .zip tanpa ekstrak.

	Perintah : unzip -l [nama_file.zip]
	Fungsi   : Perintah tersebut akan melihat isi file .zip tanpa harus ekstrak terlebih dahulu.


=== > 7z

01. Ekstrak file .7z ke folder saat ini.

	Perintah : 7z x [nama_file.7z]
	Fungsi   : Perintah tersebut akan mengekstrak isi file .7z ke folder saat ini.

02. Ektsrak file .7z ke folder lain.

	Perintah : 7z x [nama_file.7z] -olokasi/folder/output
	Fungsi   : Perintah tersebut akan mengekstrak isi file .7z ke folder lain yang akan dituju.
	Catatan  : Setelah mengetikkan -o tidak boleh ada spasi.

03. Melihat isi file .7z tanpa ekstrak.

   	Perintah : 7z l [nama_file.7z]
	Fungsi   : Perintah tersebut akan melihat isi file .7z tanpa harus ekstrak terlebih dahulu.


=== > unrar

01. Ekstrak file .rar ke folder saat ini.

	Perintah : unrar x [nama_file.rar]
	Fungsi   : Perintah tersebut akan mengekstrak isi file .rar ke folder saat ini.

02. Ekstrak file .rar ke folder lain.

	Perintah : unrar x [nama_file.rar] namaFolder/
	Fungsi   : Perintah tersebut akan mengekstrak isi file .rar ke folder lain yang akan dituju.

03. Melihat isi file .rar tanpa ekstrak.

	Perintah : unrar l [nama_file.rar]
	Fungsi   : Perintah tersebut akan melihat isi file .rar tanpa harus ekstrak terlebih dahulu.
